Section 4: Automations to Build

Build these automation workflows in Kit Commerce. Each one has specific triggers, filters, and email sequences.

1. AP Primary Nurture
Trigger: Tag "rate-calculator-user" is applied
Filter: NOT authenticity-premium-buyer
Sequence: 5-email AP nurture sequence
Exit Condition: Apply tag "ap-sequence-complete-no-buy" after all 5 emails sent
Purpose: Introduce AP to rate calculator users who haven't bought yet
Build AP Primary Nurture automation
2. AP Post-Purchase
Trigger: Tag "authenticity-premium-buyer" is applied
Filter: None
Sequence: 4-email post-purchase sequence
Exit Condition: After all 4 emails sent
Purpose: Cross-sell Closer Kit to AP buyers. Provide implementation tips, success stories
Build AP Post-Purchase automation
3. Kit Post-Purchase
Trigger: Tag "first-1k-kit-buyer" is applied
Filter: None
Sequence: 4-email post-purchase sequence
Exit Condition: After all 4 emails sent
Purpose: Ascend Kit buyers to AP. Show how AP expands on Kit foundations
Build Kit Post-Purchase automation
4. Closer Post-Purchase
Trigger: Tag "ugc-closer-kit-buyer" is applied
Filter: None
Sequence: 3-email post-purchase sequence
Exit Condition: After all 3 emails sent
Purpose: Cross-sell AP if they don't already own it. Provide community access invite
Build Closer Post-Purchase automation
5. Cart Abandonment - AP
Trigger: Tag "ap-cart-abandon" is applied (manual or via pixel)
Filter: None
Sequence: 3-email cart recovery sequence
Exit Condition: After all 3 emails sent or purchase completed
Purpose: Recover abandoned AP sales. Include testimonials, limited-time incentive
Build Cart Abandonment - AP automation
6. Cart Abandonment - Kit
Trigger: Tag "kit-cart-abandon" is applied
Filter: None
Sequence: 2-email cart recovery sequence
Exit Condition: After all 2 emails sent or purchase completed
Purpose: Recover abandoned Kit sales. Lower price point = shorter sequence
Build Cart Abandonment - Kit automation
7. Cart Abandonment - Closer
Trigger: Tag "closer-cart-abandon" is applied
Filter: None
Sequence: 2-email cart recovery sequence
Exit Condition: After all 2 emails sent or purchase completed
Purpose: Recover abandoned Closer Kit sales
Build Cart Abandonment - Closer automation
8. Win-Back Campaign
Trigger: Tag "ap-sequence-complete-no-buy" is applied
Filter: No open/click activity in last 14 days
Delay: 14 days after sequence completes
Sequence: 4-email win-back sequence
Purpose: Re-engage inactive subscribers. Use scarcity, objection handling, new angles
Build Win-Back Campaign automation
9. Cash Spike Flash Sales
Type: Manual broadcast (not automated)
Audience: Non-buyer segments (rate-calculator-user NOT in any buyer tag)
Purpose: Time-limited promotions to drive quick conversions
Frequency: Use strategically (1-2 times per quarter recommended)

Section 5: Order Bump Setup

Configure order bumps to increase average order value. Start with AP → Kit bump.

Order Bump: Kit Add-On to AP
Location: AP checkout page (final step, before payment)
Product: First $1K Kit
Price: $27 (normally $47)
Copy: "Add the First $1K Kit - 60 negotiation scripts + 100-brand database for just $27 (normally $47)"
Placement: Full-width card above "Complete Purchase" button
Success Tag: Automatically apply both "authenticity-premium-buyer" and "first-1k-kit-buyer" if accepted
Configure AP → Kit order bump
Order Bump Best Practices:
- Keep copy short and benefit-focused
- Use social proof (testimonials, # of users)
- Make it a clear visual element (don't hide)
- Ensure smooth checkout flow if they accept
- Test mobile experience carefully
- Monitor acceptance rate; refine copy if under 10%
Plan future order bumps
Consider: Kit → AP upsell, Closer → AP bundle, limited-time addon products
